Inicio

Nisse Først es un bot con especificas tipos de funciones,
interacción entre miembros, comandos divertidos y funciones para staff.
A su vez, hace parte del proyecto Nisse-Little.

TIPOS DE COMANDOS

Interacción

El sistema de interacción van a hacer un servidor mas activo,
algunos de ellos requieren de mencionar un usuario. Mientras
otros se pueden realizar por si mismos; los comandos se enviaran
en un formato de embed junto a un gif aleatorio

Diversión

Esta sección trae todo tipo de comandos, la mayoria randomizados
algunos pueden ser comandos de azar o confesiones
también pueden hacerse con menciones de miembros.

Moderación

Los comandos de moderación traen y agregan funciones externas que
mejoran la experiencia de los miembros del staff.Un ejemplo claro
puede ser el comando ns!kick. Con este comando
se pueden (obviamente) expulsar miembros, sin embargo se puede añadir una
razón que será enviada en un mensaje embed al usuario expulsado (o baneado).
